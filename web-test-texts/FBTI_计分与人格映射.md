# 计分与人格映射（27人格版）

> 版本：2026年6月性格面板同步版  
> 配套问卷：16题  
> 配套人格：27种足球搭子人格

---

## 一、计分规则

每道题每个选项包含三部分：

- `primary`：主推人格，每命中一次 +2 分
- `secondary`：次推人格，每命中一次 +1 分
- `dimensions`：题目选项的性格维度倾向，用于隐藏款触发判断和题目设计校验

最终角色排序使用：

1. 角色原始得分
2. `ROLE_SCORE_BIAS` 概率偏置
3. primary 命中次数
4. `ROLE_ORDER` 固定顺序兜底

注意：结果页雷达面板使用 `ROLE_DIMENSIONS` 中的角色固定性格分数，不直接由答题动态生成。

## 二、维度说明

| 维度 | 当前显示名 | 含义 |
|---|---|---|
| E | 上头指数 | 看球时情绪表达和瞬间反应强度 |
| R | 战术画板 | 关注战术、结构、技术逻辑和理性分析 |
| S | 群聊活跃 | 看球时互动、开麦、接梗和参与讨论的倾向 |
| N | 戏精程度 | 对故事性、戏剧性、名场面和人设的偏好 |
| V | 投机心态 | 对冒险、冷门、心理战和不确定性的接受度 |
| G | 跟风系数 | 受群体氛围、看台声量和舆论节奏影响的程度 |

## 三、当前映射源

当前题目、选项、主推人格、次推人格和维度倾向已经统一写入：

- `app.js` 的 `QUESTIONS`
- `app_demo.js` 的 `QUESTIONS`

文案展示版见：

- `web-test-texts/FBTI_题目与选项.md`

这版映射的设计原则是让题目指向贴合新版雷达面板：

- 战术/数据选项主要指向 `R` 高的角色：`PEP`、`DAT-A`、`TINTIN`、`CARD-MA`、`LEO`、`NEY-MAR`
- 巨星/关键球选项主要指向 `E/R/N/V` 高的角色：`CR7`、`8-PE`、`LEO`、`NEY-MAR`
- 群聊/陪伴选项主要指向 `S/G/E` 高的角色：`LAO-8`、`HEI-HEI`、`CN-12`、`TUI-Q`、`FAN-ZY`
- 叙事/名场面选项主要指向 `N` 高的角色：`WAN-SUI`、`CCTV-HE`、`1100`、`LEGEND`、`ZI-DANE`、`DINHO`
- 心理战/冒险选项主要指向 `V` 高的角色：`MA-DING`、`LE-FU`、`MOU`、`Z-LATAN`、`ZI-DANE`

## 四、隐藏人格触发条件

| 隐藏人格 | 触发条件 |
|---|---|
| `LEGEND` 传奇 | 进入前5 + Q8选C + Q12选D + Q15选A + N维度 >= 85 |
| `ZI-DANE` 齐祖 | 进入前5 + Q9选D + Q10选B + Q15选B + V维度 >= 85 |
| `DINHO` 小罗 | 进入前5 + Q9选D + Q14选B + Q15选D + V维度 >= 85 |

隐藏款仍然受 `ROLE_SCORE_BIAS` 压低总概率；即使选项命中，也需要满足隐藏触发条件才会稳定出现。

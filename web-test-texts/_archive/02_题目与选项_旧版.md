一些IDEAS：
1.我们可以做"省流版"（8题）和"详细版"16题，让不想答题的人也能快速有一个分类；
2.答完省流 8 题后显示初步人格卡片 + 可分享，并提示"还有 8 题可以解锁稀有人格款（铁子/淇楚·王/李惠堂/伊基塔）"作为 milestone 激励
3.隐藏人格触发后的卡片可以有差异化标识（比如金边/特殊底色）——这是二次传播的强引擎，"我抽到隐藏款了"本身就是社交货币。
4.做一个互补人格映射+和matchmate的衔接文案，例如：
"这是你的人格。下面是几种最适合和你一起看球的搭子。" 
• 列 2-3 个互补人格 + 一句话说明为什么互补 
• 然后接："真人不容易遇到？matchmate 直接给你一个（链接/小程序入口）"

## 三、节奏曲线

整套问卷设计成一段心理曲线：

```
省流段（破冰+核心识别）              进阶段（深度+隐藏触发）
─────────────────────────  ▶  ────────────────────────
   Q1-Q5 + Q8-Q9 + Q14                Q6-Q7 + Q10-Q13 + Q15-Q16
   "进入状态" + "看见自己"             "解锁稀有款"

8 题做完 = 可分享结果 + milestone 转化点 → 进阶 8 题 = 解锁隐藏款
```

- **Q1-Q5**：从国足/比赛/价值观入门，密度高
- **Q8-Q9**：自我转向（AI 题藏在这一段）+ 直接人格映射
- **Q14**：自我画像收尾（省流段终点）
- **Q6-Q7 + Q10-Q13**：进阶题考立场、瞬间偏好、群聊互动
- **Q15-Q16**：进阶段补隐藏款 signature——共情/沉默爆发 + 历史敬意

2.手机适配：移动端怎么做才能在主流场景（微信、小红书）里顺畅跑？  
3.分享/截图通道：怎么让用户主动愿意分享，并且觉得"发这张截图超有意思"？
